Details

Final project for FEND Degree aligned with the real needs, and hopefully the result can be useful for some people. ESPA is an European funding program which, among other, provides preschool sponsorship for parents with low income in Greece. Every year the catalogues of nurseries and kindergartens are published on the website as pdf, and as a parent you spend quite some time, googling every address in your area trying to find if it's located close to you. The backend script parses this PDF files and geocodes all the addresses, exporting a nice-formatted json with only useful information. This json is loaded on front end to Google Map, showing all the locations together with data on the ages of children, operation hours, etc. The app also attempts to search for the page of kindergarten on Facebook, and if it exists, shows some useful information from it.
