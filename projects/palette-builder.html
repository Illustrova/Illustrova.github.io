Details

This is a quick tool I wrote for myself to create, import and export palette following Opium.Fill naming convention. It currently supports JSON, CSS, SASS and Stylus syntax. You can read Opium.Fill concept here: https://medium.com/@opium.pro/standardisation-of-color-schemes-through-the-eyes-of-a-programmer-53cc25148470
