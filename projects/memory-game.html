Details

Had some fun research on the abilities of SVG and CSS animations, while preparing a project for my FEND Nanodegree. Instead of using generic layout, I've built my own one and added some experimental effects. Although the result, particularly card "explosion" is too slow to be considered for production, it was a great research about the abilities of SVG.
